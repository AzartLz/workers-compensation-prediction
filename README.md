# Прогнозирование стоимости страховых выплат (Workers Compensation)

Данный проект представляет собой интерактивное веб-приложение на базе **Streamlit** для анализа данных о производственных травмах и прогнозирования итоговой стоимости страхового возмещения (`UltimateIncurredClaimCost`).

##  Цель проекта
Разработка и деплой модели машинного обучения, которая помогает страховым компаниям более точно оценивать финансовые резервы, необходимые для выплат пострадавшим работникам.

##  Технологический стек
* **Язык:** Python 3.10+
* **Библиотеки ML:** Scikit-learn, XGBoost, NumPy, Pandas
* **Интерфейс:** Streamlit
* **Презентация:** Streamlit-reveal-slides (на базе Reveal.js)
* **Визуализация:** Matplotlib, Seaborn

##  Структура проекта
* `app.py` — точка входа в приложение, настройка навигации.
* `analysis_and_model.py` — страница с загрузкой данных, их предобработкой, обучением моделей и формой для предсказания.
* `presentation.py` — интерактивная бизнес-презентация проекта.
* `requirements.txt` — список зависимостей для установки.
* `data/` — папка для хранения локальных наборов данных.

##  Запуск проекта

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/AzartLz/workers-compensation-prediction.git](https://github.com/AzartLz/workers-compensation-prediction.git)
    cd workers-compensation-prediction
    ```

2.  **Установите необходимые библиотеки:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Запустите Streamlit-приложение:**
    ```bash
    streamlit run app.py
    ```

##  Описание работы
1.  **Предобработка:** В приложении реализована очистка данных, извлечение признаков из дат (день недели, месяц, задержка отчета) и масштабирование числовых переменных.
2.  **Моделирование:** Проводится сравнение моделей Linear Regression, Ridge и Random Forest.
3.  **Интерактив:** Пользователь может ввести параметры страхового случая (возраст, зарплата, начальная оценка) и мгновенно получить прогноз итоговой стоимости.
