# Проект: Прогнозирование стоимости страховых выплат

## Описание проекта

Цель проекта — разработка модели машинного обучения для предсказания итоговой стоимости страхового возмещения (UltimateIncurredClaimCost).

## Датасет

Используется датасет Workers Compensation (OpenML, ID: 42876), содержащий 100 000 записей.

## Технологии

- Python
- pandas
- scikit-learn
- matplotlib
- seaborn
- streamlit
- xgboost

## Установка

```bash
pip install -r requirements.txt

streamlit run app.py
